# Section 1: Formal Email Phrases (ビジネスメール)

## 1.1 Opening Phrases (頭語)

| Usage | Phrase | Context |
|-------|--------|---------|
| First contact | 突然のご連絡失礼いたします | 初対面のメール |
| Regular correspondent | お世話になっております | 取引・継続的な関係 |
| After break | ご無沙汰しております | 久々の連絡 |
| Formal beginning | 拝啓 | 極めて硬い文書 |

**Example:**
```
拝啓
時下ますますご清栄のこととお慶び申し上げます。
突然のご連絡失礼いたしました。
...
敬具
```

## 1.2 Cushion Words (クッション言葉)

Cushion words soften requests and obligations.

### Request Patterns
| Japanese | English equivalent |
|----------|-------------------|
| よろしければ | if it's convenient |
| 可能であれば | if possible |
| お忙しいところ恐れ入ります | I know you're busy, but... |
| 勝手ながら | if you'll allow me |
| お時間がある際に | when you have time |

**Example:**
```
お忙しいところ恐れ入りますが、
来週の水曜日までに資料のご確認をお願いします。
```

### For Requests (お願い)
- お忙しいところ、恐縮ですが
- せっかくですが
- お忙しい中とは存じますが
- 恐れ入りますが

### For Refusals (丁重ご辞退)
- 、せっかくですが
- あいにく
- 誠に残念ながら
- お気持ちはありがたいのですが

### For Apologies (お詫び)
- 大変恐縮ですが
- 私事で恐縮ですが
- 申し上げにくいのですが

# Section 2: Workplace Keigo (ビジネス敬語)

## 2.1 Three Types of Keigo

### Respect Honorifics (尊敬語)
Used for the addressee's or third party's actions.
```
行く → いらっしゃる、おいでになる
来る → いらっしゃる，お越しになる
言う → おっしゃる
食べる → 召し上がる
```

### Humble Honorifics (謙譲語)
Used for one's own actions to show respect to the listener.
```
行く → 伺う，参る
来る → 伺う，参る
言う → 申す，申し上げる
食べる → いただく，顶戴する
```

### Polite Language (丁寧語)
Used to make the entire sentence polite.
```
です，ます，ございます
お+，ご+
```

## 2.2 Common Keigo Conversions

| Plain | Respectful | Humble | Polite |
|-------|------------|-------|--------|
| 行く | いらっしゃる | 伺う，参る | 行きます |
| 来る | 見える，お越しになる | 参る | 来ます |
| する | なさる | いたします | します |
| 言う | おっしゃる | 申す | 言っています |
| 見る | 見られる、ご覧になる | 拝見する | 見ています |
| 聞く | お聞かせになる | 伺う、承る | 聞いています |
| 食べる | 召しが上がる | いただく | 食べます |
| 知っている | 存じる | 存じる | 知っています |
| 给我 | 给我 | 差し上げる | あげます |

## 2.3 Common Mistakes to Avoid

### Double Keigo (二重敬語)
❌ Wrong: 伺わせていただきます
○ Correct: お伺いします

❌ Wrong: 拝見させていただきます
○ Correct: 拝見いたします

### Inappropriate Respect
❌ Wrong: ご苦労様です (to seniors)
○ Correct: お疲れ様です

❌ Wrong: 了解しました (to seniors)
○ Correct: 承知いたしました，かしこまりました

# Section 3: Senior-Junior Communication

## 3.1 How Juniors Speak to Seniors

### Basic Rules
- Use polite forms (です・ます)
- Use humble honorifics for own actions
- Use respectful honorifics for seniors' actions
- Avoid casual language

### Common Expressions
| Situation | Expression |
|-----------|-------------|
| Greeting | おはようございます |
| Asking | ○○様のお時間をいただいてもよろしいでしょうか |
| Requesting | ○○をお出しいただけますでしょうか |
| Thanking | ありがとうございます |
| Apologizing | 申し訳ございません |

## 3.2 How Seniors Speak to Juniors

Seniors typically use:
- だ・だよね casual forms with close juniors
- Standard polite forms (です・ます) in formal situations
- ってください (imperative) for instructions
- だな for confirmation

### Example Patterns
```
部長 → 部下A：この資料、帮我确认一下，30分钟后给我。
部下A → 部长：好的了解。现在开始确认。

部长 → 部下A：好的。我明白了。I'll let you know in 30 minutes.
```

# Section 5: Phone/Meeting Conversation Patterns

## 5.1 Phone Conversation

### Receiving Calls
1. Answer within 3 rings
2. Greeting: お電話ありがとうございます。○○株式会社の△△でございます。
3. Ask for caller identity: 恐れ入りますが、お名前，您的公司名称是？

### When Party is Absent
```
对方：田中さん，请问在吗？

我方：对不起，现在出去了。几点左右回来？

对方：大概3点左右。

我方：回来后给您回电话可以吗？您的电话号码是？
```

|-----------|-------------|

# Section 7: Differences from Casual Keigo

## 7.1 Key Differences

| Aspect | Casual Keigo | Business Keigo |
|--------|--------------|----------------|
| Speech style | More casual | Full keigo |
| Context | Social | Formal business |
| Email | Possible | Required |
| Self-reference | 僕/俺/あたし | 私/わたくし |

## 7.2 Formal Adaptations

### Casual → Business
- ゆく → 行く → おいでになる，いらっしゃる
- もらう → いただく，頂戴する
- やる → 差し上げる
- あげげる → 差し上げる

### Additional Politeness Markers
- お忙しいところ
- 恐れ入ります
- 申し訳ございません
- ご迷惑いたしました

## 7.3 Honorifics Nuances

### 御社 vs 貴社 vs 社
- 御社：口语/演讲中使用
- 貴社��书��文件中使用
- 社：组合词（如「入社」）中使用

### 部長 vs A氏長
对外说明本公司上司时：
❌ Wrong：我们的部长
○ Correct：我们的总经理
（对外提到本公司人员时应使用职位，不加敬称）

# Key Takeaways

1. **Always use polite forms** - business environments require です・ます consistently
2. **Use humble forms for yourself** - いただく, 伺う, 申す
3. **Use respectful forms for others** - なさる, 見える, 見られる
4. **Avoid double keigo** - never stack keigo forms
5. **Match the situation** - formal emails require full礼状文, phone calls require different patterns than in-person
6. **Be careful with titles** - don't use honorifics when referring to your own company's superiors to outsiders
7. **Watch industry-specific terms** - each industry has unique terminology
